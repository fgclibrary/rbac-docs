本节包含权限管理框架所有对外 API 的详细说明。接口基于 OpenAPI 3.1 规范定义，支持以下功能：

- **用户管理** — 用户列表查询、用户详情获取、用户创建与删除
- **角色管理** — 角色列表查询、角色创建
- **组织管理** — 组织列表查询、组织创建
- **权限管理** — 角色权限查询与更新
- **元数据管理** — 元数据查询与更新
