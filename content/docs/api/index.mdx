本节包含权限管理框架所有对外 API 的详细说明。接口通过「服务端命令」提供对外服务，接口定义基于 OpenAPI 3.1 规范，包含以下功能模块：

- **用户管理** — 用户列表查询、用户详情获取、用户创建与删除
- **角色管理** — 角色列表查询、角色创建
- **组织管理** — 组织列表查询、组织创建
