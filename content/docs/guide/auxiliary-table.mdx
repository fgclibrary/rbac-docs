框架构建了 4 个辅助表，用于配合各模块完成业务逻辑。

## 标签页头字典表

服务于角色模块的前端页面，为标签页头提供动态显示数据。

<Callout>
框架实际仅使用 ID 为 7 和 8 的标签页头。如需自定义该表，请确保页头 ID 与页面 `roleMembership` 的标签点击命令逻辑保持一致。
</Callout>

| 字段名  | 类型 | 说明   |
|:-----|:---|------|
| ID   | 主键 | 唯一标识 |
| 编码   | 文本 |      |
| 名称   | 文本 |      |
| 使用范围 | 文本 |      |
| 描述   | 文本 |      |

## 角色树形控件图标表

服务于角色模块的前端页面，为角色树提供图标数据。其 `ID` 字段与 `角色分类关系视图` 中的自定义公式字段 `图标 ID` 关联。

| 字段名 | 类型 | 说明                        |
|:----|:---|---------------------------|
| ID  | 主键 | 唯一标识                      |
| 图片  | 图片 | 图标存储路径标识，格式为 `GUID_<文件名>` |

## 组织树形控件图标表

服务于组织模块的前端页面，为组织树提供图标数据。其 `ID` 字段与 `组织信息视图` 中的自定义公式字段 `图标` 关联。

| 字段名 | 类型 | 说明                        |
|:----|:---|---------------------------|
| ID  | 主键 | 唯一标识                      |
| 图片  | 图片 | 图标存储路径标识，格式为 `GUID_<文件名>` |

## 有效性字典表

服务于用户模块，提供账户有效性的文本映射。页面 `accountList` 的表格中，`账户状态` 列通过公式与该表的 `名称` 隐性关联。

| 字段名 | 类型 | 说明           |
|:----|:---|--------------|
| ID  | 主键 | 唯一标识         |
| 有效性 | 数值 | 0 表示失效，1表示有效 |
| 名称  | 文本 |              |
